# FC모바일 카드 제작기 (FC Mobile Card Creator)

FC모바일 카드를 쉽게 만들고 공유할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 카드 제작 도구
- 자유게시판
- 카드 공유 및 좋아요 기능
- 이미지 업로드 및 관리

## 기술 스택

- HTML5
- CSS3
- JavaScript
- Firebase (Firestore, Storage)

## 설치 및 실행

1. 저장소 클론
```bash
git clone [repository-url]
```

2. 웹 서버 실행
```bash
# 로컬 웹 서버 실행 (예: Python)
python -m http.server 8000
```

3. 브라우저에서 접속
```
http://localhost:8000
```
